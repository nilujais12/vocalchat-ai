# 🤖 VocalChat AI – Chatbot & Voice Assistant

VocalChat AI is a browser-based AI assistant built using HTML, CSS, and JavaScript. It supports voice commands, speech output, and basic task automation, providing an interactive chatbot experience directly in the browser.

---

## 🚀 Features

- 🎤 Voice input using Web Speech API (Speech-to-Text)
- 🔊 Speech output using Text-to-Speech
- 💬 Interactive chatbot interface
- ⏰ Time and date information
- ⏳ Timer functionality
- 🧮 Basic mathematical calculations
- 😂 Random joke generation
- 🌐 Open popular websites (YouTube, Google, etc.)
- 🔎 Google search integration
- ⚡ Real-time response system

---

## 🛠️ Technologies Used

### 🔹 HTML
- Structured the chatbot interface (chat window, input field, buttons)
- Integrated microphone button for voice commands

### 🔹 CSS
- Designed responsive and user-friendly UI
- Styled chat messages, layout, and overall appearance

### 🔹 JavaScript
- Implemented core chatbot logic and command handling
- Used **Web Speech API**:
  - `SpeechRecognition` for voice input
  - `SpeechSynthesis` for voice output
- Applied **event handling** for user interactions
- Implemented features like:
  - Time & date responses
  - Timer functionality
  - Web navigation
  - Basic calculations
  - Joke responses

### 🔹 Browser APIs
- `navigator.mediaDevices.getUserMedia()` for microphone access
- Window object for opening websites and navigation
